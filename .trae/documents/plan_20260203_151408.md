# GitHub Upload & Setup Guide

## 1. Prepare Files

Since the automated file organization was skipped, please run these commands in your terminal to structure the project:

```bash
# Create the Dria directory
mkdir -p Dria

# Rename and move the script
mv install_dria_optimized.sh Dria/install_dria.sh
chmod +x Dria/install_dria.sh
```

## 2. Initialize Git & Commit

Initialize the repository and commit your files:

```bash
git init
git add .
git commit -m "Initial commit: Add Dria installer and README"
```

## 3. Upload to GitHub

1. Create a new repository named `deploy_nodes` on GitHub.
2. Link your local repository and push (replace `<your-username>`):

```bash
git branch -M main
git remote add origin https://github.com/<your-username>/deploy_nodes.git
git push -u origin main
```

## 4. Usage

Once pushed, anyone can install using this one-liner (update the username):

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/<your-username>/deploy_nodes/main/Dria/install_dria.sh)
```
